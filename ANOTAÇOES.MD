Repositório
É a pasta do projeto.


commit
È um grupo de alterações no código


Branch

São ramificações do projeto, serve para separa grandes
modificações e  evitam criar problemas no código principal.



Merge

União de duas branches

Clone

È como uma branch de de repositorio online em uma repositório local.
Quando se deseja trabalha em um repositório hospedado no github,
clona-se esse repositório para o seu computador, trabalho se nele e 
então é pedido a permissão para atualizar as alterações online.

Pull

É a atualizaçÕ do repositório online.


Fork
È como um clone, porém dentro do github. isso quer dizer que o 
repositór não vai ser baixado paea o seu computador, mas será 
criado um ingual em sua conta.

Pull Request

É um pedido que se faz ao dono do repositório para que esse atualize 
ocodigo dele com o seu código. Ou seja, voçê pede para que o dono do projeto 
ao qual voçê quer contribuir adicionando suas modificações ao peojeto oficial.
